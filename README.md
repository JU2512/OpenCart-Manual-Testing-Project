#  OpenCart Manual Testing Project

A comprehensive **Manual Testing Project** for the OpenCart eCommerce platform focused on validating core functionalities, identifying defects, and documenting structured QA processes.

---

#  Project Objective

To perform end-to-end manual testing on OpenCart by designing test scenarios, executing test cases, identifying bugs, and ensuring application quality across major user workflows.

---

#  Scope of Testing

This project covers functional validation of key OpenCart modules such as:

* User Registration
* Login & Logout
* Product Search
* Product Details Page
* Add to Cart
* Shopping Cart
* Checkout Process
* Order Placement
* Contact Form
* UI & Navigation Validation
